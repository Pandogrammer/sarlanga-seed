    $( "#connect" ).click(function() { connect(); });

       cuando toco el boton, activa

JS:    stompClient.send("/app/hello", {}, JSON.stringify({'name': $("#name").val()}));

       le cae a

JAVA:  @MessageMapping("/hello")

       y lo devuelve en 

JAVA:  @SendTo("/topic/greetings")

       que lo ataja por estar subscripto a

JS:    stompClient.subscribe('/topic/greetings', function (greeting) {
          showGreeting(JSON.parse(greeting.body).content);
       });

       lo manda al html mediante

JS:	   function showGreeting(message) {
    	  $("#greetings").append("<tr><td>" + message + "</td></tr>");
	   }

	   que lo muestra en

HTML:  <tbody id="greetings">
